# Review Questions

## What is Node.js?
- A JS run-time environment that executes JS server side

## What is Express?
- An HTTP server framework for Node

## Mention two parts of Express that you learned about this week.
- How to use it to perform CRUD and REST operations

## What is Middleware?
- Enhancement functionality that is added to our express server

## What is a Resource?
- A route endpoint path on the API

## What can the API return to help clients know if a request was successful?
- HTTP status codes

## How can we partition our application into sub-applications?
- use export module functionality

## What is CORS and why do we need it?
- Cross Origin Resource Sharing. It allows web application to be able to connect to servers when domains are not the same.
